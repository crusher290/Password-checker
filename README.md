# **Password Checker**

>## *What is this program going to do for you?*
>> * This program asks you for a password and then tests it against 8 security filters and then scores your password based on these 8 filters.
>>> 1. The password must be longer than 8 characters.
>>> 2. Password must contain at least one `English` letter.
>>> 3. Password must contain at least one special character (`@,#,$,%,&,!,*`).
>>> 4. The password must contain at least one uppercase `English` letter.
>>> 5. The password should not be the same as the username.
>>> 6. The password should not be the `swapcae` version of the username.
>>> 7. The password should not be the same as the username, but with alternate letters. exampl's: (`a to @`),(`s to $`),(`i to !`),(`o to 0`)
>>> 8. The password should not be like common passwords. examol's: (`1234`),(`admin`),(`asdfg`),(`12345678`)
